UnityEditorExtensions

About

Open sourced package for extending the standard set of capabilities offerred by Unity game engine editor.
Contributions welcome! See Contributions for more info.

Using the Tag and Layer properties

This package introduces the Tag and Layer properties that you can use on your class's fields as shown below: 
    public class Demo : MonoBehaviour
    {
        [Tag] public string tagAttribute;
        [Layer] public int layerAttribute;
    }
This instructs the Unity Inspector to display Tag and Layer dropdowns instead of a plain string or integer text boxes for the tagAttribute and layerAttribute fields respectively, as shown below:
